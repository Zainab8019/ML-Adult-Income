# Adult Income Classification - Machine Learning Project

## Description

Ce projet de Machine Learning vise à prédire si le revenu annuel d’un individu dépasse 50K dollars à partir du dataset Adult Income.

Plusieurs algorithmes de classification supervisée ont été implémentés, comparés et évalués afin d’identifier le modèle le plus performant.

## Dataset

Adult Income Dataset

Variable cible :

* income (>50K ou <=50K)

## Algorithmes utilisés

* K-Nearest Neighbors (KNN)
* Decision Tree
* Logistic Regression
* Random Forest

## Prétraitement

* Gestion des valeurs manquantes
* Encodage des variables catégorielles
* Normalisation des données
* Séparation Train/Test

## Évaluation

* Accuracy
* Matrice de confusion
* Courbe ROC
* Validation croisée
* Importance des variables
